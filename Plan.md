📋 High-Level Feature List
1. 🌍 World Map Home Screen

App opens as an animated game world with different "lands"
Each subject = one land (Number Land, ABC Forest, Gujarati Jungle, Maths Kingdom)
Lands unlock as child progresses
A pet companion follows the child across the map


2. 🔢 Number Land (1–100)

Animated number learning with audio
Count objects game
Missing number puzzles
Odd/Even sorter
Number train ordering game
Balloon pop challenge


3. 🔤 ABC Forest (A–Z English)

Letter tracing with finger
Word + image + sound per letter (A for Apple)
Simple phrases and sentences
Alphabet karaoke song
Letter match & word builder games
Phonics sound challenge


4. 🔡 Gujarati Jungle (ગુજરાતી બારાખડી)

Full vowel (સ્વર) learning first
Row-by-row Barakhadi with tap-to-hear audio
Character tracing with sparkle trail
Listen & Tap game
Barakhadi Bingo
Voice repeat challenge (speak into mic)
Syllable to word matching


5. ➕ Maths Kingdom (+ − × ÷)

Visual block addition & subtraction
"Share the candies" division game
Times table with rhythmic chant
Spin & Solve wheel game
Math Duel vs cartoon character
Treasure chest unlock on correct answer


6. 🌟 Reward & Gamification System

Stars & coins per correct answer
Daily login streak with fire emoji 🔥
Trophy room with badges
Level titles: Seedling → Explorer → Champion → Genius
Confetti + sound on level up
Pet companion grows & evolves with points
Dress up / feed the pet (Tamagotchi style)


7. 📅 Daily Challenge Mode

5-question mixed quiz every day
Weekend bonus mega game
Streak rewards for 7, 14, 30 days


8. 👨‍👩‍👧 Parent Dashboard (Separate PIN-locked section)

Time spent per subject
Weak areas report
Daily/weekly progress chart
Set screen time limits
Enable or disable modules
Language preference setting


9. ⚙️ Settings & Personalization

Pick avatar + name on first launch
Choose companion pet
Font size control
Sound ON/OFF
Gujarati / English / Hindi toggle for instructions
Offline mode (all core content works without internet)


10. 🎨 Design Language

Bright playful colors, large tap targets
Rounded bold fonts (Baloo, Noto Sans Gujarati)
Gujarati cultural themes — kites, garba, rangoli, matki
No text-heavy screens — icons + audio do the talking
Portrait lock, phone + tablet responsive
